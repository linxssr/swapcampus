## 数据库表一览

| 表名 | 说明 |
|------|------|
| `user` | 用户表（学号/昵称/密码/信用分/状态） |
| `item` | 商品表（标题/价格/分类/封面/审核状态） |
| `item_image` | 商品图片表（一商品多图） |
| `item_collect` | 收藏表（用户-商品关联） |
| `category` | 商品分类表 |
| `orders` | 订单表（买卖双方/交易方式/状态） |
| `comment` | 评价表（评分/内容） |
| `chat_message` | 聊天消息表（发送方/接收方/内容/已读） |
| `report` | 举报记录表（被举报用户/原因/处理结果） |
| `admin` | 管理员表（独立账号体系） |
| `browse_history` | 浏览历史表（首页推荐） |
| `sign_record` | 签到记录表（日期/连续天数） |
| `task_config` | 任务配置表（任务名/奖励/类型，静态配置） |
| `user_task` | 用户任务进度表（完成/待领取/已领取状态） |
| `student_real` | 学号真实姓名对照表（注册实名验证） |
