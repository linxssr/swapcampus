1. WebSocket 连接接口
接口：
WS /ws/chat?token=xxx
实现文件：
backend/module-m5-chat/src/main/java/com/campus/chat/websocket/ChatServer.java
backend/module-m5-chat/src/main/java/com/campus/chat/config/ChatWebSocketConfig.java
功能：
• 建立 WebSocket 连接
• 从 token 中解析用户 ID
• 维护在线用户连接
• 接收消息 JSON
• 保存消息到数据库
• 点对点转发给在线接收方
• 给发送方回传已保存消息
• 连接关闭时移除在线状态
• 连接异常时自动清理
当前开发阶段为了方便测试，token 支持两种形式：
/ws/chat?token=1
/ws/chat?token=2
也支持正式 JWT，后续用户登录模块做好后可以直接对接。
 
2. 聊天图片上传接口
接口：
POST /api/v1/chat/upload
实现文件：
backend/module-m5-chat/src/main/java/com/campus/chat/controller/ChatController.java
backend/module-m5-chat/src/main/java/com/campus/chat/service/ChatService.java
backend/module-m5-chat/src/main/java/com/campus/chat/service/impl/ChatServiceImpl.java
功能：
• 接收 MultipartFile
• 校验不能为空
• 校验必须是图片类型
• 上传到 MinIO 的 chat/ 目录
• 返回图片 URL
注意：这个接口需要本地 MinIO 正常运行，否则上传会失败。
 
3. 获取历史聊天记录接口
接口：
GET /api/v1/chat/history/{toUid}
实现功能：
• 从请求头 Authorization 解析当前用户 ID
• 查询当前用户和指定用户之间的历史聊天记录
• 按发送时间升序返回
• 查询时自动将对方发给当前用户的未读消息标记为已读
开发阶段测试时可以这样传请求头：
Authorization: 1
或者正式格式：
Authorization: Bearer <jwt-token>
