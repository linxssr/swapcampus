### 二、用户主页功能

#### 1. 功能概述

新增用户个人主页功能，支持查看任意用户的公开信息、发布商品和评价历史。区分两种访问场景：
- 访问自己的账号 → 跳转到完整个人中心 (`/profile`)
- 访问他人账号 → 显示公开主页 (`/user/:userId`)

#### 2. 后端新增接口

**获取用户公开信息：**
```
GET /api/v1/user/profile/{userId}
```
- 返回：`userId`、`username`、`creditScore`、`avatar`
- 无需登录，公开接口

**获取用户发布的商品：**
```
GET /api/v1/item/user/{userId}
```
- 返回指定用户发布的所有商品列表
- 无需登录，公开接口

**获取卖家评价历史：**
```
GET /api/v1/comment/seller/{sellerId}
```
- 返回该卖家收到的所有评价
- 包含：买家名称、商品标题、评分、评价内容、评价时间

**实现文件：**
- `UserController.java` - 已有 `getUserProfile()` 方法

### 三、数据库字段优化

#### CommentMapper 查询优化

**原有问题：**
- `selectByItemId()` 和 `selectBySellerId()` 返回的字段是 snake_case（`buyer_id`、`item_title`）
- 前端期望 camelCase 格式

**解决方案：**
- 使用 SQL `AS` 别名将查询结果字段转为 camelCase
- 示例：
```sql
SELECT c.comment_id AS commentId, c.buyer_id AS buyerId,
       u.username AS buyerName, i.title AS itemTitle
FROM comment c
LEFT JOIN user u ON u.user_id = c.buyer_id
LEFT JOIN item i ON i.item_id = c.item_id
```
