# SwapCampus后端六大模块接口清单&开发拆分

## 整体说明

技术：SpringBoot3 + Mybatis + JWT鉴权 + MinIO + WebSocket；分**6大必做模块**，接口区分【普通用户接口/管理员接口】，统一前缀`/api/v1`

## 一、六大模块接口清单

### M1 用户体系模块（学号实名、JWT鉴权、信用分、个人中心）


| 接口       | 请求方式 | 路径                          | 说明                            |
| -------- | ---- | --------------------------- | ----------------------------- |
| 用户注册     | POST | /api/v1/user/register       | 学号唯一注册，密码加密                   |
| 用户登录     | POST | /api/v1/user/login          | 返回JWT token，绑定用户信息            |
| 获取个人信息   | GET  | /api/v1/user/info           | JWT鉴权，返回昵称、头像、信用分(初始100)      |
| 修改个人资料   | PUT  | /api/v1/user/update         | 昵称、手机号、头像(MinIO)              |
| 修改密码     | PUT  | /api/v1/user/pwd            | 旧密码校验，新密码加密保存                 |
| 查看我的发布   | GET  | /api/v1/user/myItem         | 当前用户发布商品列表                    |
| 查看我的订单   | GET  | /api/v1/user/myOrder        | 买家订单/卖家订单分类查询                 |
| 获取用户公开信息 | GET  | /api/v1/user/profile/{userId} | 查看指定用户昵称、头像、信用分（用于个人主页）       |
| 查看用户发布商品 | GET  | /api/v1/item/user/{userId}  | 查看指定用户发布的所有商品（公开接口，无需登录）     |


**管理员独立登录**
|管理员登录|POST|/api/v1/admin/login|生成管理员专属JWT，无用户权限|

### M2 商品发布模块（多图MinIO、分类绑定、待审核）


| 接口   | 请求方式   | 路径                       | 说明                |
| ---- | ------ | ------------------------ | ----------------- |
| 图片上传 | POST   | /api/v1/item/uploadImg   | 文件上传MinIO，返回URL   |
| 发布商品 | POST   | /api/v1/item/publish     | 表单+多图URL入库，默认待审核0 |
| 编辑商品 | PUT    | /api/v1/item/update      | 仅发布者可编辑，修改后重回待审核  |
| 删除商品 | DELETE | /api/v1/item/delete/{id} | 下架删除              |


### M3 商品检索模块（分类、搜索、筛选、收藏）


| 接口     | 请求方式   | 路径                          | 说明        |
| ------ | ------ | --------------------------- | --------- |
| 全部分类列表 | GET    | /api/v1/category/list       | 首页分类下拉    |
| 分类查商品  | GET    | /api/v1/item/cate/{cid}     | 按分类查已上架商品 |
| 关键词搜索  | GET    | /api/v1/item/search?key=xx  | 标题模糊查询    |
| 高级筛选   | GET    | /api/v1/item/filter         | 价格区间、成色筛选 |
| 添加收藏   | POST   | /api/v1/collect/add         | 收藏商品      |
| 取消收藏   | DELETE | /api/v1/collect/cancel/{id} | 取消收藏      |
| 我的收藏   | GET    | /api/v1/collect/my          | 查询本人收藏列表  |


### M4 交易流程模块（下单、面交/柜取、确认收货、评价、信用变动）


| 接口       | 请求方式 | 路径                              | 说明                     |
| -------- | ---- | ------------------------------- | ---------------------- |
| 创建订单     | POST | /api/v1/order/create            | 生成唯一订单号，1面交2柜取         |
| 查询单个订单   | GET  | /api/v1/order/{oid}             | 订单详情                   |
| 买家确认收货   | PUT  | /api/v1/order/finish/{oid}      | 订单变为已完成，开启评价权限         |
| 取消订单     | PUT  | /api/v1/order/cancel/{oid}      | 未确认可取消                 |
| 新增评价     | POST | /api/v1/comment/add             | 订单完成才可评价，影响双方信用        |
| 查看商品评价   | GET  | /api/v1/comment/item/{itemId}   | 商品所有评价（含买家名、评分、内容）     |
| 查看订单评价   | GET  | /api/v1/comment/order/{orderId} | 查询指定订单的评价              |
| 查看卖家评价历史 | GET  | /api/v1/comment/seller/{sellerId} | 查询卖家收到的所有评价（含商品名、买家名） |


### M5 站内通讯模块（WebSocket实时聊天，文字/图片/表情）


| 接口       | 请求方式 | 路径                           | 说明            |
| -------- | ---- | ---------------------------- | ------------- |
| 建立聊天连接   | WS   | /ws/chat?token=xxx           | JWT校验身份，点对点通讯 |
| 聊天图片上传   | POST | /api/v1/chat/upload          | 聊天图片存入MinIO   |
| 获取历史聊天记录 | GET  | /api/v1/chat/history/{toUid} | 和指定用户历史消息     |
| 标记单条消息已读 | PUT  | /api/v1/chat/read/{msgId}    | 将当前用户收到的指定消息isRead置为1 |


### M6 管理员后台模块（审核、举报、用户封禁、分类管理）


| 接口     | 请求方式 | 路径                            | 说明          |
| ------ | ---- | ----------------------------- | ----------- |
| 商品审核   | PUT  | /api/v1/admin/item/audit      | 0待审→1通过/2驳回 |
| 用户封禁   | PUT  | /api/v1/admin/user/ban/{uid}  | 用户状态0封禁/1解封 |
| 全部举报列表 | GET  | /api/v1/admin/report/list     | 待处理/已处理筛选   |
| 处理举报   | PUT  | /api/v1/admin/report/handle   | 填写处理结果      |
| 新增分类   | POST | /api/v1/admin/category/add    | 后台新增商品分类    |
| 修改分类   | PUT  | /api/v1/admin/category/update | 编辑分类名称排序    |
